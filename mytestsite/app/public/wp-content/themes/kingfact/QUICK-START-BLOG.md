## Image Guidelines

### Featured Image
- **Minimum Size**: 800x600px
- **Recommended**: 1200x800px
- **Format**: JPG or PNG
- **File Size**: Under 500KB (compress if needed)

### Content Images
- Use the "Add Media" button in editor
- Add alt text for SEO
- Align left, right, or center as needed

## Categories vs Tags

### Categories
- **Broad topics** (e.g., Web Development, Design, Marketing)
- **Hierarchical** (can have parent/child)
- **Limited number** (5-10 recommended)
- **Required for organization**

### Tags
- **Specific keywords** (e.g., CSS, WordPress, SEO)
- **Non-hierarchical** (flat structure)
- **Unlimited** (use as many as relevant)
- **Optional but helpful**
